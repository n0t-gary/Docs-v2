# Contributing to Polytoria

Welcome to the documentation for contributing to Polytoria! This will contains the guide on how you can improve Polytoria Open source projects. Including the game client, creator and API documentation.

## Contributing to the Client/Creator

### Bug reports

If you've came across a bug. You can create a new issue on our [Github Issues](https://github.com/Polytoria/polytoria-game/issues). Before you do so, make sure you search up for existing issues first, someone might already reported it. If you're not sure, you can just submit. And we'll get them! 

### Feature requests

If you wanted to request a feature. You can use our [Github Discussions](https://github.com/Polytoria/polytoria-game/discussions/categories/ideas) page. Just like bug reports, please make sure to search up for existing discussion on it first. If there isn't any, feel free to create a new one. 

### Contributing to the source

Interested in contributing to the source? Please check out [Software Guides](/software) for guides on how to get started with the source! Additionally, you can join the [Polytoria Contributors](https://discord.gg/HUgEE9FhSU) Discord Server if you have any questions regarding the source.
